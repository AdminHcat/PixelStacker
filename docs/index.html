Pangamma's PixelStacker

A program built to make multi layer pixel art using blocks from Minecraft. It incorporates the use of stained glass stacked on top of other blocks in-game. The result is better blending and more color selections.

About the Project


It is my hope that you enjoy this program. I made this program as a proof of concept for creating high quality pixelart. This program provides a few viewing options to help you actually build the art in-game. You can also save to PNG or Schematic format. Most importantly. You can use the double layer mode which combines stained glass panels with newer Minecraft blocks to create an even larger color pallet.

Features

The Viewer

Enjoy the nice image viewer GUI. You can pan by dragging with a mouse. Zoom by scrolling

Rendered Viewer

After rendering the image (CTRL + R), you can see the blocks that are chosen for the build.

Options

Choose from all the latest and greatest Minecraft blocks for colors. Toggle side/top views, change the grid size, and even double-layer mode.

View Grid

Losing track of where you are sucks. Toggle the grid view (CTRL + G) if you want help, and if you want to adjust the grid size, you can do that in the options menu.

View Texture

Sometimes you just want to see how the program simplifies your build. You can disable textures to see the build painted with average color codes instead of texture images.

View Outline

Seeing the outline helps if you're working with partially transparent images. It helps you visualize the boundaries of your pixel art.

Double Layer Mode

By using stained glass panes you can create all kinds of color combinations. It makes a huge difference because you are essentially multipling the available color pool by 16. The results are impressive!

View Output


WorldEdit Origin Editor

Double click to set the WorldEdit origin. The origin will show up as the red dot. If you try to paste a schematic, the schematic will be pasted relative to the red dot.

Save and Export

You can save as a schematic, or as a PNG file. Schematics will always face north. Ideal for dynmaps.

Pallette Viewer

Show a hybrid view that combines texture mode and smooth mode. Useful for if you want to see what average color is being used for the matching algorithm.

Quantizer Engine

Drastically improve rendering speed by reducing the total number of colors used in the rendering process. You can use this for some cool effects. Many algorithms to choose from.


Dithering

If you are a fan of dithering, then you will enjoy choosing between 15 different dithering algorithms.

Show Case
